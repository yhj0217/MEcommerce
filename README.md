## 👖 프로젝트 소개

- 머리부터 발끝까지! 모자, 옷, 액세서리, 신발을 모두 구매할 수 있는 쇼핑몰 사이트입니다.

## 👟 주요 기능

- 로그인 / 로그아웃 / 회원가입
- 홈 / 카테고리 / 카테고리별 상품 상세
- 판매 상품 등록 / 등록 상품 전체 조회 / 수정 / 삭제
- 상품 장바구니 추가 / 수정 / 삭제
- 장바구니 내 상품 주문 / 결제
- 구매 내역 / 구매 취소
- 판매 내역 / 판매 상태 변경
